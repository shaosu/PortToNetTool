# PortToNet

## 功能
1. 基础串口调试
2. 支持数据通过网络双向转发
3. 支持CAN数据转发

3. Prism.DryIoc 8.1.97
4. HandyControls 3.5.2
5. 无边框窗体,自带菜单
6. 实现主题切换,语言切换
7. 实现IconFont字体
